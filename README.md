perfctl — это утилита командной строки для Linux, позволяющая переключать профили производительности системы с помощью горячих клавиш. Скрипт интегрируется с powerprofilesctl и обеспечивает быстрый доступ к режимам Power Saver, Balanced и Performance, аналогично фирменным сочетаниям клавиш на ноутбуках Lenovo.

Возможности:
Мгновенная смена профиля производительности без открытия терминала.
Поддержка назначения горячих клавиш через системные средства (например, gnome-keybinding, xbindkeys, sxhkd).
Отображение уведомлений о текущем профиле (через notify-send или аналогичные механизмы).
Минимальный и нативный интерфейс, без лишних зависимостей.

Установка:
git clone https://github.com/CyberMaloy/perfctl.git
cd perfctl/
cp perfctl /usr/sbin/perfctl
chown root:root /usr/sbin/perfctl
chmod 755 /usr/sbin/perfctl

Далее добавляем горячие клавиши в GNOME/KDE которые будут использовать команду perfctl тем самым изменяя профиль производительности.
